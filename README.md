# Retro-Service.eu 🚗📟

Retro-Service is a web-based management platform designed for vintage vehicle enthusiasts and collectors. It helps users track maintenance history, manage spare parts, and document the restoration process of retro technology.

[![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg)]()
[![Framework](https://img.shields.io/badge/Framework-Laravel%2012-orange.svg)](https://laravel.com)
[![Frontend](https://img.shields.io/badge/Frontend-Livewire%203-fb70a9.svg)](https://livewire.laravel.com)

## 🚀 Technical Stack

- **Backend:** PHP 8.3 + Laravel 12 (Eloquent, Migrations, Service Classes)
- **Frontend:** Livewire 3 (Reactive UI), Alpine.js, Blade
- **Database:** MySQL / MariaDB
- **Styling:** Bootstrap / SCSS
- **Tools:** Composer, NPM, Git

## ✨ Key Features

- **Service Management:** Detailed logs for vehicle maintenance and repairs.
- **Parts Inventory:** Keep track of available and needed spare parts.
- **Modern UI:** A fast, reactive interface for a seamless user experience across devices.
- **Search & Filter:** Instant filtering of records using Livewire components.
